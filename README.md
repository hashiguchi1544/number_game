# リポジトリ名
https://github.com/hashiguchi1544/number_game

## アプリ名
数当てゲーム

## どのようなアプリか
1~50の中にランダムな正解の数が１つあり、それを当てていくげゲーム

## 作成した理由
JavaScriptの学習にため。MDNのJavaScriptのチュートリアルで学習していたところ、それを改良できるのではにないかと思い込んアプリを作成。

## 学習したもの
HTML, CSS, JavaScript
